What is it ?

Competitive programming is a mind sport for solving coding problems using algorithms and data structures. The participants need to write code under various restrictions like memory limits, execution time, the limit of coding, space, etc. The winner of competitive programming is declared based on problems solved and time spent for writing successful programming solutions. However, it also includes other factors like quality of output produced, execution time, program size, etc.
